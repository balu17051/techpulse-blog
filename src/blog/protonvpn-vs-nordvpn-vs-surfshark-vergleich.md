## Datenschutz & Jurisdiktion: Das Kleingedruckte

### ProtonVPN — Schweiz 🇨🇭
- **Sitz**: Plan-les-Ouates, Genf
- **Gesetzgebung**: Schweizer Datenschutzgesetz (DSG) + Bundesgesetz über die Überwachung des Post- und Fernmeldeverkehrs (BÜPF)
- **Vorteil**: Keine EU-Datenschutzrichtlinie, keine 14-Eyes-Mitgliedschaft, starke Whistleblower-Gesetze
- **Nachteil**: BÜPF erlaubt gezielte Überwachung mit Richterbeschluss — aber **keine Massenüberwachung**
- **Audit**: Securitum (2022), Mozilla (2023) — Open Source Client + Server-Code einsehbar
- **Transparenzbericht**: Jährlich, sehr detailliert

### NordVPN — Panama 🇵🇦
- **Sitz**: Panama City
- **Gesetzgebung**: Keine Vorratsdatenspeicherung, keine 5/9/14-Eyes-Mitgliedschaft
- **Vorteil**: Panama hat **keine Gesetze** die VPNs zur Datenspeicherung zwingen
- **Nachteil**: Firma hinter NordVPN (Tefincom S.A.) undurchsichtig, Verbindungen zu Tesonet (Litauen) — Kritik an Unternehmensstruktur
- **Audit**: Deloitte (2022, 2023, 2024) — No-Logs bestätigt, Infrastruktur geprüft
- **Bug Bounty**: Aktives Programm, hohe Auszahlungen

### Surfshark — Niederlande 🇳🇱
- **Sitz**: Amsterdam (Surfshark B.V.)
- **Gesetzgebung**: EU-Mitglied, DSGVO, **9-Eyes-Land** (Niederlande teilen Geheimdienstinfos)
- **Vorteil**: DSGVO-Schutz, Deloitte-Audit (2023), RAM-only Server
- **Nachteil**: 9-Eyes — theoretisch Zugriff durch NL-Geheimdienst + Partner
- **Audit**: Deloitte (2023), Cure53 (Browser-Extension)
- **Übernahme**: 2022 von Nord Security (NordVPN-Mutter) gekauft — operiert aber eigenständig

**Datenschutz-Sieger**: **ProtonVPN** (Schweiz + Open Source + Transparenz). **NordVPN** nah dran (Panama + Audits). **Surfshark** wegen 9-Eyes und Eigentümerstruktur schwächer — aber für 99% der Nutzer irrelevant.

## Spezial-Features im Detail

### NordVPN: Threat Protection Pro
Nicht nur Ad-Block — **Malware-Scanner für Downloads**, **Tracker-Blocker**, **URL-Filter**. Funktioniert **systemweit** (nicht nur Browser). Ersetzt teilweise Antivirus. In Standard-Plan enthalten.

```bash
# Linux CLI Beispiel
nordvpn connect Germany
nordvpn set technology nordlynx
nordvpn set killswitch on
nordvpn set threatprotection on
```

### Surfshark: Alternative ID & Alert
- **Alternative ID**: Generiert Fake-Identität (Name, Adresse, E-Mail) für Registrierungen — Datenschutz bei Newsletter/Foren
- **Alert**: E-Mail/Kreditkarten-Monitoring bei Datenlecks (HaveIBeenPwned-Integration)
- **CleanWeb**: Ad/Tracker/Malware-Block auf DNS-Ebene
- **Bypasser**: Split Tunneling für Apps/Websites

### ProtonVPN: Secure Core & Stealth
- **Secure Core**: Traffic routed durch **Schweiz/Island/Schweden** (Datenschutz-Havens) bevor er Exit-Server erreicht — schützt vor kompromittierten Exit-Nodes
- **Stealth Protokoll**: Verschleiert VPN-Traffic als HTTPS — umgeht Deep Packet Inspection (China, Iran, Russland, Firmen-Firewalls)
- **Tor over VPN**: Ein Klick → .onion-Sites im normalen Browser
- **NetShield**: DNS-Ad-Block (Free: Basis, Plus: Aggressiv)

## Häufige Fragen (FAQ)

### "Ist ein kostenloses VPN okay?"
**Nein.** Kostenlose VPNs finanzieren sich über **Datenverkauf**, **Bandwidth-Weiterverkauf** (dein Gerät wird Exit-Node für Andere), **Malware**. ProtonVPN Free ist die **einzige Ausnahme** — aber: 3 Länder, 1 Gerät, kein Streaming, kein P2P, langsamer. Für ernsthafte Nutzung: Zahle die €2-4/Monat.

### "Kann ich VPN auf dem Router installieren?"
**Ja, und du solltest.** Router-VPN schützt **alle Geräte** (Smart TV, Konsole, IoT, Gäste-WLAN) ohne App-Installation.
- **Einfachste**: NordVPN Native App (GL.iNet, Asus, DD-WRT)
- **Manuell**: WireGuard/OpenVPN Config auf jedem OpenWrt/GL.iNet/Asus

### "Verlangsamt VPN mein Internet?"
**Ja, 10-30% Overhead** (Verschlüsselung, Routing). Bei 250 Mbit/s Leitung: ~180-220 Mbit/s — für **alles** (4K Streaming, Gaming, Downloads) **mehr als genug**. Nur bei <50 Mbit/s Basis merkt man es.

### "Ist WireGuard sicher?"
**Ja.** Moderne Kryptografie (ChaCha20-Poly1305, Curve25519, BLAKE2s), ~4.000 Zeilen Code (vs. 100.000+ OpenVPN), formale Verifikation. **Standard 2026**. Alle drei nutzen WireGuard (NordVPN: NordLynx-Fork).

### "Brauche ich Double VPN / Multi-Hop?"
**Für 99%: Nein.** Einmal verschlüsselt reicht. Double VPN halbiert Speed, erhöht Latenz. Nützlich nur bei: **hochwertige Targets** (Journalisten, Aktivisten, Whistleblower) wo Exit-Node-Kompromittierung realistisch ist.

### "Funktioniert VPN in China/Russland/Iran?"
**Nur mit Obfuscation/Stealth.**
- **NordVPN**: Obfuscated Server (funktioniert oft)
- **Surfshark**: NoBorders Modus (hit-or-miss)
- **ProtonVPN**: **Stealth Protokoll (beste Chance)** — entwickelt für Zensur-Umgehung
